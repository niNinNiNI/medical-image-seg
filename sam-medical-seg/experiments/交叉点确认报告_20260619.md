### 🔑 核心发现

**交叉点精确位于 5% → 10% 之间。**

- 1%: SAM-MedSeg 领先 4.0pp
- 5%: SAM-MedSeg 领先收窄至 2.4pp  
- 10%: U-Net 反超 2.5pp ← **交叉点**
- 100%: U-Net 大比分领先 7.9pp（均值 0.719 vs 0.640）

这意味着：**当标注数据达到 80 张（10%）时，传统 U-Net 已经超越 SAM-MedSeg。** SAM-MedSeg 的核心价值聚焦在 ≤5%（≤40 张）的极端少样本场景。

## 五、论文叙事建议（更新版）

### 核心主张

> **SAM-MedSeg 是极端少样本（≤5% 标注数据）医学图像分割的最优方案。** 当标注数据极度稀缺时，基于 SAM 预训练 + LoRA 微调的方案显著优于传统 U-Net。但当标注数据达到 10%（80 张）以上时，传统方法开始反超。这一定量化的交叉点为论文提供了清晰的实用指导。

### 实验支撑

| 论点 | 证据 |
|------|------|
| 零样本 SAM 不可用 | Dice 0.271, Precision 0.175 |
| ≤5% 少样本场景 SAM-MedSeg 最优 | 1%: +4.0pp, 5%: +2.4pp vs U-Net |
| **交叉点在 5%~10% 之间** | **10%: U-Net +2.5pp（反超）** 🆕 |
| 充足数据下传统方法更优 | U-Net 100%: 0.723 vs SAM-MedSeg: 0.640 |
| LoRA 是极端少样本的核心驱动力 | 227K 参数在 5% 达 0.452 Dice |
